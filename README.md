# CQC Registration Tracker

A clean, mobile-friendly React + TypeScript checklist app for CQC registration planning.

## Supabase setup

This app is wired for Supabase, but auth is not enabled yet.

Create a `.env.local` file with:

```bash
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```

If you want me to link a live Supabase project next, I need:

1. Your Supabase project ref
2. Your Supabase project URL
3. Your Supabase anon key
4. A Supabase access token if you want CLI linking to the hosted project

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
